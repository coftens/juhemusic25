### 1. 即刻天气 (Jike)
该接口最为稳定，响应速度快，且对文件格式校验较松。

- 请求地址 : http://www.jiketianqi.com/weatapi/file/upload
- 请求方法 : POST
- 数据格式 : multipart/form-data
- 关键参数 :
  - files : 文件对象 (Key 名必须为 files )
- 必要 Header :
  - Referer : http://www.jiketianqi.com/
  - Origin : http://www.jiketianqi.com
- 返回结果 :
  - 格式：JSON
  - 路径： data[0] (返回的是一个数组，取第一个元素即为 URL)
### 2. 石油气 (Oil)
该接口支持 HTTPS，适合对安全性有要求的场景，且上传逻辑非常直接。

- 请求地址 : https://cn.oilgasgpts.com/file/upload
- 请求方法 : POST
- 数据格式 : multipart/form-data
- 关键参数 :
  - file : 文件对象 (Key 名必须为 file )
- 必要 Header :
  - Referer : https://cn.oilgasgpts.com/feedback
  - Origin : https://cn.oilgasgpts.com
- 返回结果 :
  - 格式：JSON
  - 路径： msg (字段名为 msg ，内容即为完整的图片 URL)
